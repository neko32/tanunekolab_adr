# Design System: Tanunekolab Cool Theme

## 1. Visual Theme & Atmosphere

**Overall mood:** Balanced and professional

**Density:** Moderate. The layout uses a varied spacing scale.

**Depth:** Flat design with minimal shadow usage.

## 2. Color Palette & Roles

- **White** (`#FFFFFF`) — Page background
- **Blue** (`#7297C5`) — Accent background
- **Black** (`#000000`) — Primary text
- **Dark gray** (`#666666`) — Secondary text
- **Light gray** (`#E5E5E5`) — Divider / border
- **Black** (`#1A1A1A`) — Divider / border

## 3. Typography Rules

**Primary font:** -apple-system
**Secondary font:** Times New Roman

**Type scale:**
- Headings: 24px, 32px, 48px, 60px
- Body / UI: 14px, 15px, 16px, 20px

**Weights in use:** 300, 400, 600, 700

**Line heights:** 27.2px, 25.5px, 32px, 40.8px, 23.8px, 81.6px, 34px, 66px

**Letter spacing:** -0.24px, 0.15px, 0.32px, -0.96px, -0.4px, -1.8px

## 4. Component Stylings

### Buttons
- Text color: `rgb(114, 151, 197)`
- Padding: 14px 32px 14px 32px
- Font: 15px, weight 400

### Cards
- Background: `rgb(255, 255, 255)`
- Padding: 0px 0px 0px 0px

### Inputs
- Border color: `#E5E5E5`

## 5. Layout Principles

**Spacing scale:** 12px, 14px, 16px, 24px, 28px, 32px, 40px, 120px

**Base unit:** 16px grid (values are mostly multiples of 16).

## 6. Guidelines

### Do
- Use 16px grid for all spacing
- Use `#7297C5` for primary actions and CTAs
- Stick to 4 font weights: 300, 400, 600, 700
- Use `-apple-system` as the primary typeface

### Don't
- Don't introduce colors outside the palette above
- Don't mix fonts beyond -apple-system and Times New Roman
- Don't use inline styles when the design system covers the pattern
